# Fast Trade Bot

Telegram бот для торговых сигналов криптовалют.

## Деплой на Railway

1. Форкните репозиторий
2. Создайте проект на [Railway](https://railway.app)
3. Подключите репозиторий
4. Добавьте переменную окружения:
   - `BOT_TOKEN` — токен вашего Telegram бота

Railway автоматически задеплоит бот.

## Локальный запуск

```bash
# Установка зависимостей
pip install -r requirements.txt

# Создайте .env файл
cp .env.example .env
# Отредактируйте .env и добавьте BOT_TOKEN

# Запуск
python bot.py
```

## Команды бота

- `/start` — Главное меню
- `/positions` — Активные позиции
- `/stats` — Статистика
- `/help` — Помощь

## Функции

- Торговые сигналы (BTC, ETH, BNB, SOL)
- Позиции с P&L в реальном времени
- Технический анализ
- Статистика сделок
